# OpenSCAD Parametric Mechanical Design Portfolio

This repository contains a collection of parametric mechanical design projects developed using **OpenSCAD**, focusing on **engineering-driven design**, **modularity**, and **manufacturing readiness**.

All models are built using **Constructive Solid Geometry (CSG)** principles and structured scripting approaches to ensure flexibility, reusability, and efficient design iteration.

---

## 🔧 Project Overview

### 1. Parametric Filleted L-Bracket
- Fully parametric geometry (length, width, thickness)
- Filleted edges using Minkowski operations
- Mounting hole customization
- Designed for structural use and 3D printing

---

### 2. Universal Joint (U-Joint)
- Functional rotational joint mechanism
- Parametric axis and connection geometry
- Designed with motion capability in mind
- Modular structure for reuse

---

### 3. Linear Bearing Block
- Parametric housing for linear motion systems
- Bearing seat and mounting interfaces
- Designed considering alignment and load transfer
- Suitable for CNC and 3D printing applications

---

### 4. Elbow Pipe with Dual Flange
- Pipe + flange integrated design
- Constant diameter flow channel
- Parametric flange dimensions and bolt holes
- Suitable for fluid systems and prototyping

---

## ⚙️ Design Approach

All designs follow a structured engineering workflow:

- **Parametric Modeling:**  
  All critical dimensions are adjustable via parameters.

- **Modular Code Structure:**  
  Reusable modules are used to improve scalability and readability.

- **CSG-Based Geometry:**  
  Complex geometries are created using union, difference, and intersection operations.

- **Manufacturing Awareness:**  
  Designs are optimized for:
  - 3D printing (FDM)
  - CNC machining
  - Assembly compatibility

## 📦 Usage

You can open and modify all models using OpenSCAD:

1. Install OpenSCAD  
2. Open any `.scad` file  
3. Adjust parameters  
4. Render (F6)  
5. Export as STL for manufacturing
